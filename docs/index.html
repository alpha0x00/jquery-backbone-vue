jQuery 的优势

  1. 提供统一的 API
  2. 创造很多高效用法
  3. 封装了错误处理

jQuery 的问题

  1. 只关注 DOM

Backbone

Backbone 的优势

  1. MVP 框架
  2. 将业务抽象成数据对象
  3. 视图响应事件更新

Backbone 的问题

没有数据双向绑定

总结
